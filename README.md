## Cara Tambah Postingan
1. Copy salah satu file di folder `/posts`.
2. Ubah nama file (contoh: `2025-09-15-artikel-baru.html`).
3. Edit isi sesuai artikelmu.
4. Tambah link ke `index.html` di daftar postingan.
