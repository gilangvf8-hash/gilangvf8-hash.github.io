<h2>
Berikut langkah-langkah untuk membuat situs web dengan GitHub Pages:
</h2>


1. Di pojok kanan atas di GitHub, klik menu untuk New repository. 


2. Beri nama repositori dengan format username.github.io, ganti username dengan nama pengguna GitHub-mu. Contoh: jika kamu octocat, maka menjadi octocat.github.io. 


3. Pilih visibilitas repositori (public/private) sesuai kebutuhan. 


4. Aktifkan opsi Add README agar repositori dilengkapi dengan file README.md. 


5. Klik Create repository untuk membuat repositori baru. 


6. Setelah repositori dibuat, buka tab Settings (jika tidak terlihat, gunakan menu dropdown). 


7. Di menu samping (sidebar), cari bagian Code and automation, lalu klik Pages. 


8. Di bagian Build and deployment, pada opsi Source, pilih Deploy from a branch. 


9. Masih di bagian Build and deployment, pilih branch dari dropdown sebagai publishing source. 


10. (Opsional) Buka file README.md di repositorimu. Kamu bisa mengedit kontennya sekarang atau biarkan default untuk sementara. 

11. Buatlah file index.html

12. Untuk melihat situsmu, kunjungi username.github.io. Perlu diperhatikan: bisa butuh hingga 10 menit agar perubahan yang kamu dorong ke GitHub muncul dan dipublikasikan. 




---
<h2>
Mengubah Judul dan Deskripsi
</h2>
Secara default, judul situs adalah username.github.io. Kamu bisa mengubah judul dan menambahkan deskripsi melalui file _config.yml di repositori.

1. Klik tab Code di repositorimu. 


2. Cari dan klik file bernama _config.yml. 


3. Klik tombol untuk mengedit file tersebut. 


4. Di dalam _config.yml, sudah ada baris tema default. Tambahkan baris baru seperti:
<textarea>
title: gilangvf8-hash / Keruntuhan Hizbut Tahrir
description: Catatan dan artikel pribadi
theme: minima
plugins:
  - jekyll-sitemap
  - jekyll-feed
  - jekyll-seo-tag
url: "https://gilangvf8-hash.github.io"
</textarea>


Contoh (dari dokumentasi):


<textarea>
title: gilangvf8-hash / Keruntuhan Hizbut Tahrir
description: Catatan dan artikel pribadi
theme: minima
plugins:
  - jekyll-sitemap
  - jekyll-feed
  - jekyll-seo-tag
url: "https://gilangvf8-hash.github.io"
</textarea>



5. Setelah selesai mengedit, klik Commit changes untuk menyimpan perubahan.

Langkah Selanjutnya (Next Steps)

Setelah berhasil membuat, mempersonalisasi, dan mempublikasikan situs GitHub Pages pertamamu, berikut beberapa hal yang bisa dieksplor lebih lanjut:

Menambahkan konten (halaman tambahan) ke situsmu menggunakan Jekyll. 

Menggunakan domain khusus/custom domain untuk situsmu, agar tidak hanya menggunakan github.io.
