<p>Berikut <b>cara menampilkan postingan (artikel) ke halaman index GitHub Pages</b>, supaya muncul di <b>beranda (index)</b> situs kamu.</p>

<p>Ada 3 cara, tergantung apakah kamu memakai <b>Jekyll tema Minima</b>, <b>tema lain</b>, atau <b>manual HTML</b>.</p>

<h2>âœ” 1. Jika Kamu Menggunakan Jekyll Blog (misalnya tema Minima)</h2>

<p><code>index.md</code> atau <code>index.html</code> milik tema Jekyll biasanya otomatis menampilkan daftar postingan dari folder <code>_posts/</code>.</p>

<h3>Langkah-langkah:</h3>

<h3>1ï¸âƒ£ Buat postingan di folder <code>_posts/</code></h3>

<p>Format wajib:</p>
<pre><code>_posts/2025-11-23-nama-artikel.md
</code></pre>

<p>Contoh isi:</p>
<pre><code>---
layout: post
title: "Cara Menampilkan Postingan di Index GitHub"
date: 2025-11-23
---

Isi artikel kamu di sini.
</code></pre>

<h3>2ï¸âƒ£ Index otomatis menampilkan postingan</h3>

<p>Jika kamu menggunakan <b>tema Minima</b>, halaman index sudah memiliki kode built-in:</p>

<pre><code>&lt;ul&gt;
  {% for post in site.posts %}
    &lt;li&gt;
      &lt;a href="{{ post.url }}"&gt;{{ post.title }}&lt;/a&gt;
    &lt;/li&gt;
  {% endfor %}
&lt;/ul&gt;
</code></pre>

<p>Sehingga semua postingan tampil otomatis di beranda.</p>

<h2>âœ” 2. Jika Index Kamu Kosong / Tidak Menampilkan Postingan</h2>

<p>Jika <code>index.md</code> atau <code>index.html</code> kamu tidak menampilkan postingan, tambahkan kode ini:</p>

<h3>index.md</h3>
<pre><code>---
layout: default
title: Beranda
---

<h1>Postingan Terbaru</h1>

<ul>
  {% for post in site.posts %}
    <li>
      <a href="{{ post.url }}">{{ post.title }}</a> â€“ {{ post.date | date: "%d %b %Y" }}
    </li>
  {% endfor %}
</ul>
</code></pre>

<p>Atau versi HTML:</p>

<pre><code>&lt;h1&gt;Postingan Terbaru&lt;/h1&gt;
&lt;ul&gt;
  {% for post in site.posts %}
    &lt;li&gt;
      &lt;a href="{{ post.url }}"&gt;{{ post.title }}&lt;/a&gt;
    &lt;/li&gt;
  {% endfor %}
&lt;/ul&gt;
</code></pre>

<p>Setelah commit, GitHub Pages akan rebuild dan postingan muncul di index.</p>

<h2>âœ” 3. Jika Kamu Tidak Mau Pakai Jekyll, Pakai HTML Manual</h2>

<p>Kalau <b>GitHub Pages tanpa Jekyll</b>, maka kamu harus mengedit <code>index.html</code> secara manual.</p>

<p>Contoh:</p>

<h3>index.html</h3>
<pre><code>&lt;h1&gt;Postingan Saya&lt;/h1&gt;

&lt;ul&gt;
  &lt;li&gt;&lt;a href="artikel1.html"&gt;Artikel 1&lt;/a&gt;&lt;/li&gt;
  &lt;li&gt;&lt;a href="artikel2.html"&gt;Artikel 2&lt;/a&gt;&lt;/li&gt;
  &lt;li&gt;&lt;a href="artikel3.html"&gt;Artikel 3&lt;/a&gt;&lt;/li&gt;
&lt;/ul&gt;
</code></pre>
